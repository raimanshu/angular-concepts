Rounding of number upto 2 decimal places

- Math.round(num * 100) / 100

- parseFloat("123.456").toFixed(2)

Converting git repo into in vs vode structure

Open any git repo in browser -> add 1s at the end of 'github' before .com

Example - https://github.com/raimanshu/html_css_projects -> https://github1s.com/raimanshu/html_css_projects
