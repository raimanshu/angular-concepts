Implementation :-

1) Declare a variable in ts file

2) Assign a value to the declared variable in ts
declared_variable = assign_a_value ;
3) put the declared variable in html file
{{declared_variable}}

Example :-

variable_1 = "This is 'variable_1' value";
{{variable_1}}
