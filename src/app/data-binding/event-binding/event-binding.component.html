Event Binding

Intro :-

Points to remember :-

- To transfer data from template
- What is $event - special variable in angular to send data of that DOM event that was raised
- We cannot use other variable like varable, text etc or $variable, $text etc in place of $event. We get "" inside the function.

Implementation :-

1) create a function inside ts file
{{'function_name() {}'}}
2) put an event name inside small brackets in html file
(event_name)
3) Assing the function to event mentioned in html
{{"(event_name)=\'function_name()\'"}}
4) pass $event inside function_name as argument
{{"(event_name)=\'function_name($event)\'"}}
5) include this expression inside opening tag of html
